📥 Cómo descargar tus datos de Spotify

  1. Inicia sesión en Spotify desde un navegador en este enlace.
  2. Desplázate hacia abajo hasta la sección "Descarga tus datos".
  3. Solicita tu información, seleccionando el tipo de datos que quieres (historial de reproducción, datos de cuenta, etc.).
  4. Spotify te enviará un correo electrónico con un enlace para descargar tus datos una vez estén listos (puede tardar hasta 30 días).
